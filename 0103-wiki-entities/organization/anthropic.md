## 基本信息

Anthropic 是一家 AI 安全公司，开发了 Claude 系列大语言模型，在 Agent 研究和实践领域具有重要影响力。

## 在 Agent 领域的贡献

- **架构方法论**：区分 Workflow 和 Agent，提出五种 workflow 模式和自主 agent
- **实践指南**：《Building Effective AI Agents》系统阐述 agent 构建最佳实践
- **工具标准化**：推出 Model Context Protocol（MCP）标准化工具集成接口
- **基准测试**：SWE-bench agent 在 GitHub issue 解决任务上展示自主编码能力
- **核心概念**：提出 ACI（Agent-Computer Interface）、Tool Engineering 等方法论

## 相关产品

- Claude（大语言模型系列）
- Claude Agent SDK
- Model Context Protocol（MCP）
- SWE-bench agent
