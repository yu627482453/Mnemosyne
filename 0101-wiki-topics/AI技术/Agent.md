# Agent 主题综述

## 范围

Agent（智能体）主题域涵盖 LLM-based 自主代理的架构设计、运行时管理、工具编排和安全模型。核心关注如何将 LLM 推理能力与执行环境有效解耦，构建可靠、可扩展的长时间运行代理系统。

## 子主题

- Managed Agents — 托管式代理服务的架构设计
- Harness Design — 代理循环与上下文管理
- Sandbox & Security — 代码执行环境与安全隔离
- Context Engineering — 上下文窗口管理与持久化
