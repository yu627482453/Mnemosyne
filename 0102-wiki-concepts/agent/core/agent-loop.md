## 定义

Agent Loop 是 LLM 自主控制流程的核心机制。

**Workflow vs Agent**：
- Workflow：预定义路径
- Agent：动态控制

## 使用场景

✅ 编程任务、复杂探索性任务
❌ 可预定义流程（如客服）
