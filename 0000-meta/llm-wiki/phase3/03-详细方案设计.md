### 主题目录内部结构

```
{编号}-{主题名}/
├── INDEX.md                         # 本目录文件清单（名称+≤200字含义）
├── _index.md                        # 主题内知识索引（Dataview FROM 本目录）
├── _counter.md                      # ID 注册表
└── {编号}-{序号}-{slug}.md          # 知识条目
```

### 编号规则

| 千位 | 领域 | 示例 |
|------|------|------|
| 0 | 系统/元数据 | `0000-meta`、`0100-wiki-meta` |
| 1-2 | 个人生活 | `1101-健康`、`1201-投资理财` |
| 3 | AI 技术 | `3001-深度学习`、`3002-LLM` |
| 4 | 工作业务 | `4001-项目管理` |
| 5+ | 通用基础 | `5001-计算机科学`、`5101-物理学` |

### 命名规范

| 类型 | 格式 | 示例 |
|------|------|------|
| 知识条目 | `{编号}-{5位序号}-{slug}.md` | `3001-00001-transformer.md` |
| 主题文件清单 | `INDEX.md` | `3001-深度学习/INDEX.md` |
| 主题知识索引 | `_index.md` | `3001-深度学习/_index.md` |
| ID 注册表 | `_counter.md` | `3001-深度学习/_counter.md` |
| 收件箱条目 | `{日期}-{slug}.md` | `2026-05-15-新发现.md` |
| MOC 页面 | `{领域}-{类型}索引.md` | `AI技术-概念索引.md` |
| Base 视图 | `{领域}-{过滤}.base` | `AI技术-概念.base` |
| 模板文件 | `t-{类型}.md` | `t-concept.md` |
| 提示词文件 | `prompt-{操作}.md` | `prompt-ingest.md` |
| 日志文件 | `LOG-YYYY-MM-DD.md` | `LOG-2026-05-15.md` |

### 索引系统（三层+清单）

| 层 | 位置 | 方式 | 内容 |
|----|------|------|------|
| 全局元索引 | `0100-wiki-meta/INDEX.md` | 手动维护 | 主题目录列表 + 条目数 + 链接 |
| 主题文件清单 | `{目录}/INDEX.md` | 手动/Claude 维护 | 本目录所有文件名称 + ≤200字含义 |
| 主题知识索引 | `{目录}/_index.md` | Dataview FROM 本目录 | 按 type 分组的可查询知识列表 |
| Base 导航 | `0105-wiki-base/*.base` | Base GUI | 按 type/tag/domain 跨主题过滤 |
